This program simulates a rental service for a variety of vehicles.
Bikes, cars, and scooters can be rented and the program keeps
track of when these vehicles need maintenance and the money made
from the rentals.

To run the program simply run make on the command line and then
run ./car_rental.

make clean removes the executable. Make clear removes the executable
as well as any object files.
